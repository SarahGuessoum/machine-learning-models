# Machine Learning Models

A collection of machine learning models implemented in Python, developed as part of a University of Manchester Machine Learning project.

This project explores regression and classification techniques using traditional machine learning methods and deep learning, with a focus on model evaluation, optimisation and robustness.

---

## Project Overview

The repository includes implementations of:

- Linear Regression using Huber Loss and Gradient Descent
- Multi-Layer Perceptron (MLP) Neural Network
- Convolutional Neural Network (CNN) for image classification
- Model evaluation and performance comparison
- Robustness testing using noisy and corrupted datasets

---

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- PyTorch
- Matplotlib
- Jupyter Notebook

---

## Key Features

### Linear Regression
- Implemented gradient descent optimisation using Huber Loss.
- Evaluated model performance using Root Mean Squared Error (RMSE).

### Neural Networks
- Developed a Multi-Layer Perceptron (MLP) for classification tasks.
- Investigated the effect of different hyperparameters on model performance.

### Convolutional Neural Networks
- Built and trained a CNN for FashionMNIST image classification.
- Evaluated classification performance using Accuracy and Macro F1 Score.

### Robustness Analysis
- Investigated model performance under varying levels of image corruption and noise.
- Compared the robustness of different architectures.
